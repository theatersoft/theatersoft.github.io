Open Source Smart Home Platform

Theatersoft Home

Live camera feeds

Device control and sensor monitoring

Web clients securely served from your own server

Web push notifications

Bus network enables transparent connections between distributed host servers and web clients

Open source web stack

Theatersoft Home is tuned for power and performance, using well known open source JavaScript components:

  • Node.js: services and device integration
  • Redux: state management
  • Preact: fast view rendering
  • PostCSS + Stylus + CSS Modules: sane and powerful CSS
  • Rollup: efficient ECMAScript module bundling

Open device ecosystems

Use of thousands of existing devices from popular multivendor hardware ecosystems, such as Z-Wave.
