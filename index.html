Open source smart home platform

Theatersoft Home is built on a open source Javascript web stack, tuned for power and performance:

  • Node.js: services and device integration
  • Redux: state management
  • Preact: fast view rendering
  • PostCSS + Stylus + CSS Modules: sane and powerful CSS
  • Rollup: efficient ECMAScript module bundling

Open device ecosystems

Use of thousands of existing devices from popular multivendor hardware ecosystems, such as Z-Wave.
